## Layering Notes

dang qian fen ceng bu tong yi：
- **Productmo kuai、Cartmo kuai**：zou wan zheng de Route → Controller → Service → Model si ceng
- **ren zheng mo kuai、Usermo kuai、Health Check**：ye wu luo ji zhi jie xie zai routes/ wen jian zhong，wu Controller/Service ceng

## lu you zhu ce (server.js)

```
/api/health    → routes/health.js
/api/auth      → routes/auth.js
/api/user      → routes/user.js
/api/admin     → routes/admin.js
/api/products  → routes/product.js
/api/cart      → routes/cart.js
```

## Module Dependency Graph

- **routes/admin.js** → adminController → productService → Product.js + Category.js
- **routes/cart.js** → cartController → cartService → Cart.js
- **routes/product.js** → productController → productService → Product.js + Category.js
- **routes/auth.js** → zhi jie shi yong database.js + bcryptjs + jsonwebtoken
- **routes/user.js** → zhi jie shi yong database.js
- **middleware/auth.js** → bei admin/cart/product/user lu you shi yong
