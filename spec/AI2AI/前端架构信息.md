## Module Dependency Graph

- **LoginView** → auth.js(store) → api/index.js(apiAuth)
- **ProductBrowseView / ProductDetailView** → userProduct.js(store) → api/product.js(apiProduct)
- **CartView** → cart.js(store) → api/cart.js
- **FavoritesView** → userProduct.js(store) → api/product.js(apiProduct)
- **ProfileView** → auth.js(store) → api/index.js(apiUser)
- **ProductManageView** → product.js(store) → api/product.js(apiAdminProduct) + api/admin.js
- **DashboardView** → api/admin.js
- **AdminLayout** → auth.js(store) + router
