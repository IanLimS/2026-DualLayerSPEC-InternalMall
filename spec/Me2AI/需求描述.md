# Internal welfare mall Me2AI

## Project overview
This project aims to develop an internal welfare mall system，Provide internal welfare product exchange for corporate employees、Purchase and manage services。The system adopts a modern front-end and back-end separation architecture，Front-end uses Vue 3Implement a responsive interface，The backend is based on Node.js/Express.jsProvide RESTful APIServe。

## core needs

### Login module
#### User login
- Employee account login Three built-in test accounts，And a clear prompt on the login page
  - Account 1：user1/password1（Initial points 100）
  - Account 2：user2/password2（Initial points 200）
  - Account 3：user3/password3（Initial points 150）
- The login page provides the function of quickly filling in account information.
- Support enter key for quick login
- Display user name after login、department、Basic information such as position
- Stay logged in，Page refresh without loss

#### Administrator login
- Administrator account login Built-in a test account，And a clear prompt on the login page
  - account：admin/admin123
- After the administrator logs in, enter the management background，Have full system administration rights
- Support login/logout function，Ensure information security

### User functions

#### User personal information management
- View personal information：User ID、username、Mail、cell phone、integral、Role、Registration time
- Personal information editing：Support modifying email address、Phone number、Shipping address, etc.
- Avatar upload and change function
- Password modification function

#### Product browsing and search
- Product category browsing：by category（electronic、Life、Office supplies, etc.）View product
- Product list display：Contains product images、name、Points required、Stock status
- Product details page：Detailed description、Specifications、Stock quantity、Redemption instructions
- Product search function：by name、category、Search products by price range
- Product sorting function：by points、Sales volume、Sort by release time
- Product collection function：Collect products of interest

#### Shopping cart management
- Add items to cart：Support quantity selection
- Shopping cart item list：Display product information、Points required、Subtotal
- Shopping cart item quantity adjustment：Increase、reduce、delete
- Batch operation of shopping cart items：Select all、Batch delete
- Total shopping cart points：Calculate total points required in real time
- Shopping cart data persistence：Stay logged in，Sync across devices

#### Points use and management
- Points balance inquiry：Display of currently available points
- Points usage record：Exchange history、consumption time、Product information
- Points acquisition record：Sign-in rewards、Activity rewards, etc.
- Points redemption rules：Explain how to obtain points and rules for using them
- Points warning reminder：Prompt when points are insufficient

#### Order management
- Order creation：Generate order from shopping cart，Select shipping address
- Order information confirmation：Product list、Points required、Receiving information
- Order status tracking：Pending、Shipped、Completed、Canceled
- View order history：Historical order list、Order details、status update
- Order search function：by time、state、Product name search
- Order cancellation function：Unprocessed orders can be canceled，Points return

#### Shipping address management
- Shipping address list：Multiple shipping address management
- Add, delete or modify shipping address：New、edit、Delete shipping address
- Default address settings：Set default shipping address
- Address selection function：Select shipping address when placing order

### Administrator functions

#### Product management
- Product information entry：Product name、describe、category、price、Points required
- Product image upload：Support multiple image uploads、Image cropping optimization
- Product inventory management：Initial inventory setup、Stock replenishment、Stock alert
- Product on-and-off-shelf management：Products on shelves、Removed from shelves、Status management
- Product editing function：Modify product information、price、in stock
- Product deletion function：Remove items no longer available
- Product import and export：Import product information in batches、Export product list

#### Order processing management
- View order list：List of all user orders、Order filter
- View order details：Order details、User information、Product information
- Order status update：Order processing、Shipping、Completed and other status changes
- Order search function：by user、time、state、Product search order
- Order statistical analysis：Order volume statistics、Point consumption statistics
- Exception order processing：Handle abnormal orders、Points exception handling

#### User management
- View user list：List of basic information of all users
- View user details：User details、Points history、Order history
- User points management：Points adjustment、Points issuance、Points deduction
- User status management：User enabled、Disable、Role adjustment
- User search function：by username、Mail、Search users by mobile phone number
- User behavior analysis：User activity、Exchange preference analysis

#### System setup and configuration
- Product category management：Product category settings、Category sorting
- Points rule configuration：Points earning rules、Exchange rule settings
- System parameter settings：System name、Announcement information、System parameters
- Announcement management：Make an announcement、Editorial Notice、Announcement on and off the shelves

#### Statistical reports
- sales statistics：Product sales ranking、sales statistics、trend analysis
- User statistics：User growth statistics、Active user statistics
- Points statistics：Points issuance statistics、Point consumption statistics、Points balance distribution
- Order statistics：Order volume statistics、Order status distribution、Processing efficiency
- Product popularity statistics：Product views、Collection、Exchange volume statistics
